## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## How to Use

1. Upload an Image or Create New Canvas
2. Select a Tool (Pencil, Eraser, or Move)
3. Mark the Region you want to edit
4. Enter a Prompt describing your desired changes
5. Apply AI Edit
6. Download your edited image
